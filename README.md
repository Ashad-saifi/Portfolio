## ✨ Key Features

* **🎭 Dynamic Typewriter Effect:** Auto-typing text in the hero section displaying multiple developer professions.
* **🫧 Sleek Glassmorphism UI:** Built with modern CSS design patterns, including frosted-glass backgrounds, subtle borders, and smooth shadows.
* **📱 Responsive Layout:** Fully optimized for mobile, tablet, and desktop screens with an animated hamburger menu drawer.
* **🧭 Enhanced Navigation:** Custom smooth-scrolling logic that highlights the active section in the header as you scroll.
* **🎨 Micro-Animations:** Floating element cards, scale transitions on hover, and smooth gradient text accents.
* **📥 Direct CV Download:** Ready-to-use resume download linkage integrated into the action buttons.
* **💬 Contact Integrations:** Integrated messaging actions for WhatsApp, LinkedIn, and direct email.

## 🚀 How to Run Locally

Since this portfolio is built with pure HTML, CSS, and JS, running it locally is extremely simple:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Ashad-saifi/Portfolio.git
   ```
2. **Open the project folder:**
   ```bash
   cd Portfolio
   ```
3. **Launch the site:**
   * Double-click `index.html` to open it in your browser.
   * Or, run it using the **Live Server** extension in VS Code for live hot-reloading.
