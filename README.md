## 🚀 Installation & Local Run
To view the portfolio locally on your machine, follow these simple steps:

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/Ashad-saifi/Portfolio.git
   cd Portfolio
   ```

2. **Open index.html:**
   * You can open `index.html` directly in your browser, or
   * Use the **Live Server** extension in VS Code for hot-reload support.
